# Radio

---

A simple Radio Button component that allows users to select one option from several.

## Props

| Name           | Type                    | Default value    | Description                                                                                                                  |
| -----------    | ----------------------- | ---------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `selected`     | **string**, **boolean** | -                | Value of the current selected option                                                                                         |
| `onChange`     | **function**            | -                | Callback function called when the component needs to change the selected element. (value) => void                            |
| `options`      | **RadioOption[]**       | -                | Array of objects with value, label, right (optional), testId and disable (optional) props.                                   |
| `textOverflow` | **boolean**             | `false`          | If true this prop to overflows the text of the label, adding '...' and the end.                                              |
| `id`           | **string**              | -                | Id of the component                                                                                                          |
| `className`    | **string**              | -                | Adds class(es) to each option container                                                                                      |
| `style`        | **CSSProperties**       | -                | Adds style(s) to each option container                                                                                       |
