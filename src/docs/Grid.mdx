# Grid

---

The Grid should be used as the main content container. You can't use a Grid inside of another Grid, it will brake your layout.

- [Props](#Props)
- [Examples](#Examples)
- [Fluid](#Fluid)

## Props

| Name        | Type          | Default value | Description                                                                                                                             |
| ----------- | ------------- | ------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `fluid`     | **boolean**   | -             | By default, the Grid component has a specified width for any screen size. The fluid property allows the Grid to extend to a 100% width.	|
| `onClick`   | **function**  | -             | Function to call on click. This is not the best practice, but is an option if you need to use it.	                                      |
| `id`        | **string**    | -             | The Grid component id.                                                                                                                  |
| `className` | **string**    | -             | The Grid component className.                                                                                                           |
| `style`     | **object**    | -             | Object with direct style properties.                                                                                                    |
