# NavAside

---

NavAside is a sidebar navigation component that provides a convenient space for links or controls, which can be shown or hidden based on user interactions. It helps organize navigation in a clean and accessible manner.

## Props

| Name              | Type                          | Default value    | Description                                                                                  |
| ----------------- | ----------------------------- | ---------------- | -------------------------------------------------------------------------------------------- |
| `children`        | **string \| ReactNode**       | -                | Adds content to the body of NavAside.                                                        |
| `top`             | **string \| ReactNode**       | -                | Adds content to the top part of NavAside.                                                    |
| `show`            | **boolean**                   | -                | Toggles the visibility of NavAside.                                                          |
| `right`           | **boolean**                   | -                | Controls whether the Aside component appears on the right side.                              |
| `onClose`         | **() => void**                | -                | Specifies a callback function that is invoked when the NavAside is closed.                   |
| `onTransitionEnd` | **boolean**                   | -                | Specifies a callback function that is triggered when the NavAside close animation completes. |
