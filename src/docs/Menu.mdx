# Menu

---

A floating menu that should be wrapped by a relative element.

- [Props](#Props)
- [Examples](#Examples)

### Props

| Name              | Type                                                                                                                                                                                       | Default value | Description                                         |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------- | --------------------------------------------------- |
| `children`        | **ReactNode**                                                                                                                                                                              | -             | Children component passed to this container.        |
| `triggerElement`  | **ReactNode**                                                                                                                                                                              | -             | React element that triggers the Menu status.        |
| `open`            | **boolean**                                                                                                                                                                                | -             | Controls whether the Menu is visible or hidden.     |
| `id`              | **string**                                                                                                                                                                                 | -             | The id of the button.                               |
| `className`       | **string**                                                                                                                                                                                 | -             | Adds class(es) to the root container                |
| `placement`       | **left** \| **right** \| **top** \| **bottom** \| **bottom-end** \| **left-start** \| **left-end** \| **right-start** \| **right-end** \| **top-start** \| **top-end** \| **bottom-start** | -             | Controls which direction the menu should grow.      |
