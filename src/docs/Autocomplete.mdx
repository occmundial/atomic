## Props

| Name              | Type                                                                 | Default | Description       |
| ----------------- | -------------------------------------------------------------------- | ------- | ----------------- |
| textfieldProps    | **[TextField](/TextField) props:**<br /> `id`<br />`type`<br />`name`<br />`label`<br />`placeholder`<br />`selectOnFocus`<br />`clear`<br />`iconName`<br />`value`<br />`className`<br />`inputClassName`<br />`disabled`<br />`testId`<br />`disableAutoComplete`          |         | Props for the textfield |
| droplistProps     | **[Droplist](/Droplist) props:**<br /> `groups`<br />`className`<br />`filter`<br />`term`<br />`testId`            |         | Props for the droplist |
| onChange          | `function`                                            |         | onChange event for the textfield |
| onKeyUp           | `function`                                             |         | onKeyUp event for the textfield |
| onEnter           | `function`                                                |         | onEnter event for the droplist |
| onMouseDown       | `function`                                                |         | onMouseDown event for the droplist |
| onFocus           | `function`                                                         |         | onFocus event for the textfield |
| onBlur            | `function`                                                         |         | onBlur event for the textfield |
| onClear           | `function`                                                         |         | onClear event for the textfield |
| showInitialData   | `boolean`                                                            |         | Show initial data in droplist without typing |
| id                | `string`                                                             |         | The id of the component |
| className         | `string`                                                             |         | Additional className for the component |
| style             | `CSSProperties`                                                      |         | Custom styles for the component |
