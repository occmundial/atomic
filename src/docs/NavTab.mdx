# NavTab
---

The NavTop component is a versatile navigation bar designed for use at the top of web pages or applications.


## Props

| Name           | Type                                                                                                                                                                                                           | Default value    | Description                             |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------- | --------------------------------------- |
| `left`         | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Anchor href attribute                   |
| `right`        | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Anchor children                         |
| `center`       | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Anchor link attribute                   |
| `flexCenter`   | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Highlights the anchor                   |
| `top`          | **[TopProp](#TopProp)[]**                                                                                                                                                                                      | -                | Anchor onCLick event                    |
| `blue`         | **boolean**                                                                                                                                                                                                    | -                | Anchor className attribute              |
| `fixed`        | **boolean**                                                                                                                                                                                                    | -                | Anchor test id                          |
| `hideOnScroll` | **boolean**                                                                                                                                                                                                    | -                | Anchor test id                          |
| `bottom`       | **boolean**                                                                                                                                                                                                    | -                | Anchor test id                          |
| `zIndex`       | **number**                                                                                                                                                                                                     | -                | Anchor test id                          |
| `className`    | **string**                                                                                                                                                                                                     | -                | Anchor test id                          |
| `isFluid`      | **boolean**                                                                                                                                                                                                    | -                | Anchor test id                          |


### LinkElement

An object with the `type` property set to 'link', where you can use all **[NavItem props](/NavItem)** except for `children`, which should be replaced with `text`. 
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **link**                                    | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |
| `text`         | **string \| ReactNode**                     | -                | Anchor link attribute                   |

### ButtonElement

An object with the `type` property set to 'Button', where you can use all **[Button props](/Button)** except for `children`, which should be replaced with `text` and white. 
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **button**                                  | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |
| `text`         | **string \| ReactNode**                     | -                | Anchor link attribute                   |


### DropdownElement

An object with the `type` property set to 'link', where you can use all **[NavItem props](/NavItem)** except for `children`, which should be replaced with `text` and `selected`.
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **dropdownLink**                            | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |
| `text`         | **string \| ReactNode**                     | -                | Anchor link attribute                   |

### IconElement

An object with the `type` property set to 'icon', where you can use all **[NavIcon props](/NavIcon)**.
Additional NavIcon props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **icon**                                    | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |


### CustomElement

An object with the `type` property set to 'custom'.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **custom**                                  | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |
| `custom`       | **ReactNode**                               | -                | Anchor link attribute                   |
| `testId`       | **string**                                  | -                | Anchor test id                          |

### LogoElement 

An object with the `type` property set to 'logo'.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **logo**                                    | -                | Anchor href attribute                   |
| `key`          | **string \| number**                        | -                | Anchor children                         |
| `logo`         | **ReactNode**                               | -                | Anchor link attribute                   |
| `testId`       | **string**                                  | -                | Anchor test id                          |

### TopProp
An object where you can use all **[NavItem props](/NavItem)** except for `white`, `small` and `className`.
