# NavTab
---

The NavTop component is a versatile navigation bar designed for use at the top of web pages or applications.


## Props

| Name           | Type                                                                                                                                                                                                           | Default value    | Description                                                    |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------- | -------------------------------------------------------------- |
| `left`         | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Defines the components on the left side of the navigation bar  |
| `right`        | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Defines the components on the right side of the navigation bar |
| `center`       | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Defines the components on the center of the navigation bar     |
| `flexCenter`   | **([LinkElement](#LinkElement) \| [ButtonElement](#ButtonElement) \| [DropdownElement](#DropdownElement) \| [IconElement](#IconElement) \| [CustomElement](#CustomElement) \| [LogoElement](#LogoElement))[]** | -                | Defines the components of the navigation bar                   |
| `top`          | **[TopProp](#TopProp)[]**                                                                                                                                                                                      | -                | Defines the Nav Top links                                      |
| `blue`         | **boolean**                                                                                                                                                                                                    | -                | Sets blue theme                                                |
| `fixed`        | **boolean**                                                                                                                                                                                                    | -                | Sets fixed position                                            |
| `hideOnScroll` | **boolean**                                                                                                                                                                                                    | -                | Hides the nav on scroll                                        |
| `bottom`       | **boolean**                                                                                                                                                                                                    | -                | Adds bottom padding                                            |
| `zIndex`       | **number**                                                                                                                                                                                                     | -                | Specifies the nav z-index                                      |
| `className`    | **string**                                                                                                                                                                                                     | -                | Adds classes to nav container                                  |
| `isFluid`      | **boolean**                                                                                                                                                                                                    | -                | Adds a responsive padding                                      |


### LinkElement

An object with the `type` property set to 'link', where you can use all **[NavItem props](/NavItem)** except for `children`, which should be replaced with `text`. 
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **link**                                    | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |
| `text`         | **string \| ReactNode**                     | -                | Defines the child component             |

### ButtonElement

An object with the `type` property set to 'Button', where you can use all **[Button props](/Button)** except for `children`, which should be replaced with `text` and white. 
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **button**                                  | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |
| `text`         | **string \| ReactNode**                     | -                | Defines the child component             |


### DropdownElement

An object with the `type` property set to 'link', where you can use all **[NavItem props](/NavItem)** except for `children`, which should be replaced with `text` and `selected`.
Additional Button props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **dropdownLink**                            | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |
| `text`         | **string \| ReactNode**                     | -                | Defines the child component             |

### IconElement

An object with the `type` property set to 'icon', where you can use all **[NavIcon props](/NavIcon)**.
Additional NavIcon props are detailed in the table below.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **icon**                                    | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |


### CustomElement

An object with the `type` property set to 'custom'.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **custom**                                  | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |
| `custom`       | **ReactNode**                               | -                | Defines the child component             |
| `testId`       | **string**                                  | -                | Anchor test id                          |

### LogoElement 

An object with the `type` property set to 'logo'.

| Name           | Type                                        | Default value    | Description                             |
| -------------- | ------------------------------------------- | ---------------- | --------------------------------------- |
| `type`         | **logo**                                    | -                | Specifies the type of component to use  |
| `key`          | **string \| number**                        | -                | Key of the component                    |
| `logo`         | **ReactNode**                               | -                | Defines the child component             |
| `testId`       | **string**                                  | -                | Anchor test id                          |

### TopProp
An object where you can use all **[NavItem props](/NavItem)** except for `white`, `small` and `className`.
