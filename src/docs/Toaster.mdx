# Toaster

---

The Toaster component allows you to trigger any of the predefined types of alerts. Insert the Toaster component in any part of your site, and handle the alerts with the toaster object's API.

- [Props](#Props)
  - [Toaster Props](#ToasterProps)
  - [ToastLauncher Props](#ToastLauncherProps)
  - [Themes](#Themes)
  - [Others](#Others)

### Toaster Props

| Name        | Type            | Default value | Description                                                                                                                                                                                                                       |
| ----------- | --------------- | ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `container` | **DOM Element** | ``            | DOM Element to append the Toaster component into it. If you don't specify a container the Toaster will be rendered in the exact same DOM position you used it. This can in some cases generate problems with z-index or position. |

### ToastLauncher Props

| Name          | Type        | Default value | Description                                                                                                              |
| ------------- | ----------- | ------------- | ------------------------------------------------------------------------------------------------------------------------ |
| `title`       | **string**  | ``            | Toast title.                                                                                                             |
| `description` | **string**  | ``            | Toast description.                                                                                                       |
| `action`      | **Object**  | ``            | Object to render an anchor tag as a Button. Object props: label - Button text, action - function to call on button click |
| `closeIcon`   | **boolean** | ``            | With this parameter the toast will render a close icon                                                                   |
| `testId`      | **string**  | ``            | The testId property adds the data attribute data-testid to the main element and should be used for testing only.         |

<Demo title='Success' background={colors.white}>
  <SimpleToaster theme='success' />
</Demo>

```jsx
import { Toaster, toastLauncher } from "@occmundial/atomic/components";

// Success
function Component() {
  return (
    <Fragment>
      <Button
        onClick={() =>
          toastLauncher.success({
            title: "Simple toast",
          })
        }
      >
        Launch toast
      </Button>
      <Toaster container={document.body} />
    </Fragment>
  );
}
```
