# NavAvatarButton

---

Button with avatar designed to be used with the NavTab component

## Props

Props of the [Button component](/Button) are also available.

| Name         | Type                                                  | Default value    | Description                                                                |
| ------------ | ----------------------------------------------------- | ---------------- | -------------------------------------------------------------------------- |
| `noCompact`  | **boolean**                                           | -                | There are four themes available: main, secondary, white, blue, flat.       |
| `className`  | **{ avatar: string; button: string; }**               | -                | Adds classes to the avatar and button components.                          |
| `style`      | **{ avatar: CSSProperties; button: CSSProperties; }** | -                | Adds styles to the avatar and button components.                           |
| `photo`      | **boolean**                                           | -                | A url to show a photo as the avatar.                                       |
| `name`       | **boolean**                                           | -                | Pass a name to show two initials as a Avatar.                              |
