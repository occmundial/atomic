A flexible component button, with six pre-stablished themes.
The button can behave like any button with onClick event, or like an anchor with href and target.

## Props

| Name           | Type                       | Default Value | Description                                       |
|----------------|----------------------------|---------------|---------------------------------------------------|
| currentPage    | number                     |               | The current active page number                    |
| pageCount      | number                     | 10            | The total number of pages                         |
| centerPages    | number                     |               | The number of pages to display ont he center      |
| marginPages    | number                     |               | The number of pages to display on the sides       |
| previousLabel  | string                     |               | The label for the previous button                 |
| nextLabel      | string                     |               | The label for the next button                     |
| onPageChange   | (selected: number) => void |               | Callback function triggered when a page is clicked|
| hideNumbers    | boolean                    |               | Whether to hide the page numbers                  |
| breakSymbol    | string                     |               | The symbol for the break between page numbers     |
| className      | string                     |               | Additional CSS class for styling                  |
