# Colors

---

#### Atomic colors following the styleguide.

```jsx
// How to use colors
import { colors } from '@occmundial/atomic/tokens'

// Destructuring the colors object
const { sec } = colors

function Component() {
  return (
    <>
      <div style={{ background: colors.prim }} />
      <div style={{ background: sec }} />
    </>
  )
}
```
