# Avatar

---

Avatar component to represent a user. The component can display the user's photo, the user's initials or a generic icon.
It can receive an onClick or an onEdit function to handle the click action.
The onEdit function is used to handle the click action for editing the avatar and it shows a camera icon over the avatar.

## Props
| Name        | Type             | Default Value | Description                                |
|-------------|------------------|---------------|--------------------------------------------|
| `photo`     | **string**       |               | The URL of the avatar photo.               |
| `name`      | **string**       |               | The name of the user.                      |
| `size`      | **number**       | 40            | The size of the avatar in pixels.          |
| `onClick`   | **function**     |               | A function to handle the click action.     |
| `onEdit`    | **function**     |               | A function to handle the click action for editing. |
| `disabled`  | **boolean**      | false         | Determines if the avatar is disabled.      |
| `id`        | **string**       |               | The ID of the avatar.                      |
| `className` | **string**       |               | Additional CSS class for styling.          |
| `style`     | **CSSProperties**|               | Additional inline styles for styling.      |
