# NavTop

---

The NavTop `(Deprecated)` component is a versatile navigation bar designed for use at the top of web pages or applications.

## Props

| Name         | Type                       | Default value    | Description                                                                                                                    |
| ------------ | -------------------------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `top`        | **[NavItem](/NavItem)[]**  | -                | Width of the placeholder. You can use numbers or a string with a percentage.                                                   |
| `blue`       | **boolean**                | -                | Height of the placeholder. You can use numbers or a string with a percentage.                                                  |
| `isFluid`    | **boolean**                | -                | It applies a border radius of 50%. Use it with width and height to create a circle to simulate an avatar or profile picture.   |
