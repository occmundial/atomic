# MenuList

---

MenuList wrapper component.

## Props

Props of the native component are also available.

| Name             | Type                          | Default value  | Description                                                                                                                                                                       |
| ---------------- | ----------------------------- | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`       | **ReactNode**                 | -              | The content of the component.                                                                                                                                                     |                                                                                                                                      |
| `className`      | **string**                    | -              | Adds classes to the root container.        
| `padding`        | **false \| size-1 - size-12** | `size-2`       | Adds or removes padding to the root container.                                                                                                                                          |
| `dense`          | **boolean**                   | `false`        | If true, compact vertical padding designed for keyboard and mouse input is used for the list and list items. The prop is available to descendant components as the dense context. |
| `component`      | **elementType**               | `menu`         | The component used for the root node. Either a string to use a HTML element or a component.                                                                                       |
