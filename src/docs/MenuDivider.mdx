# MenuDivider

---

A simple wrapper for applying list styles to the ht tag.

## Props

| Name             | Type                     | Default value  | Description                         |
| ---------------- | ------------------------ | -------------- | ----------------------------------- |
| `className`      | **string**               | -              | Adds classes to the root container. |
| `style`          | **string**               | -              | Adds styles to the root container.  |
| `disablePadding` | **boolean**              | `false`        | If true, all padding is removed.    |
