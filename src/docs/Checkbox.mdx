# Checkbox

---

A simple Checkbox component that allows users to select an option.

## Props

| Name           | Type                    | Default value    | Description                                                                                                      |
| -------------- | ----------------------- | ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| `value`        | **boolean**             | -                | Default value for the checkbox.                                                                                  |
| `undetermined` | **boolean**             | -                | Used for partial check or when is unable to retrieve the value as absolute.                                      |
| `onChange`     | **function**            | -                | Callback function called when the component needs to change the element state. (value) => void                   |
| `disabled`     | **boolean**             | -                | Disable the toggle.                                                                                              |
| `label`        | **ReactNode**           | -                | Text to display next to the checkbox.                                                                            |
| `right`        | **string** **boolean**  | -                | Text to show on the right.                                                                                       |
| `textOverflow` | **boolean**             | -                | Use this prop to overflow the text of the label, adding '...' and the end.                                       |
| `trk`          | **boolean**             | -                | Id for catching selected element for tracking.                                                                   |
| `id`           | **string**              | -                | Id of the component                                                                                              |
| `className`    | **string**              | -                | Adds class(es) to the togglecontainer                                                                            |
| `style`        | **CSSProperties**       | -                | Adds style(s) to the toggle container                                                                            |
| `testId`       | **string**              | -                | The testId property adds the data attribute data-testid to the main element and should be used for testing only. |
