# Tooltip

---

Alert component to display messages within a container.

## Props

| Name           | Type                   | Default value | Description                                                                                                                                                                        |
| -------------- | ---------------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`     | **ReactNode**          | -             | It accepts any type of the followings `boolean`,`null`,`undefined`,`ReactFragment`,`ReactPortal`,`ReactChild`                                                                      |
| `theme`        | **string**             | `purple`      | This property can be used to set the theme of the component. The possible values for the theme property are `info`, `light`, `dark`, and `purple`.                                 |
| `show`         | **boolean**            | -             | This property can be used to control whether or not the component is displayed.                                                                                                    |
| `text`         | **string**             | -             | This property can be used to set the text that is displayed by the component.                                                                                                      |
| `openOnHover`  | **boolean**            | `false`       | Designates whether the tooltip should open on activator hover.                                                                                                                     |
| `closeDelay`   | **number**             | `4000`        | Time on millisecond that it will take to close the tooltip.                                                                                                                        |
| `zIndex`       | **number**             | `10`          | Sets the Tooltip stack order.                                                                                                                                                      |
| `placement`    | **Placement**          | `top`         | Allows to specify the placement of Tooltip. Available options are `top`, `top-start`, `top-end`, `bottom`, `bottom-start`, `bottom-end`, `left`, `right`.                          |
| `className`    | **object**             | -             | Allows to specify classes to activador and tooltip. Object has the next properties: `activator(string)`, `tooltip(string)`.                                                        |
| `showArrow`    | **boolean**            | `true`        | This property can be used to control whether or not an arrow is displayed.                                                                                                         |
| `fit`          | **boolean**            | `false`       | This property allows the Tooltip to be adjusted to the length of the associated component.                                                                                         |
| `width`        | **number/string/null** | `220`         | The "Width" property of the Tooltip enables modification of its width and accepts a number (e.g., 220), a string (e.g., '220px' or 'auto') or null as its value.                   |
| `onShowChange` | **function**           | ``            | This component allows you to change its state by toggling the tooltip open or closed. This function has the next properties: `(show: boolean) => void`. |
