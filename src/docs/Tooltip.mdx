# Tooltip

---

Alert component to display messages within a container.

## Props

| Name             | Type         | Default value | Description                                                                                                                                                                                            |
| ---------------- | ------------ | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `children`       | **ReactNode**| -             | It accepts any type of the followins `boolean`,`null`,`undefined`,`ReactFragment`,`ReactPortal`,`ReactChild`                                                                                           |
| `theme`          | **string**   | `purple`      | This property can be used to set the theme of the component. The possible values for the theme property are `info`, `warning`, `success`, `error`, `light`, `dark`, and `purple`.           |
| `show`           | **boolean**  | -             | This property can be used to control whether or not the component is displayed.                                                                                                                        |
| `text`           | **string**   | -             | This property can be used to set the text that is displayed by the component.                                                                                                                          |
| `openOnHover`    | **boolean**  | `false`       | Designates whether the tooltip should open on activator hover.                                                                                                                                         |
| `closeDelay`     | **number**   | `400`         | Time on millisecond that it will take to close the tooltip.                                                                                                                                            |
| `zIndex`         | **number**   | `10`          | Spacing between the component and the floating tooltip.                                                                                                                                                |
| `placement`      | **Placement**| `top-start`   | Placement for a component. Available options are `top-start`,`top-end`,`right-start`,`left-end`,etc.                                                                                                   |                                                                                  
| `className`      | **string**   | -             | Alert component className.                                                                                                                                                                             |
| `showArrow`      | **boolean**  | `true`        | This property can be used to control whether or not an arrow is displayed.
