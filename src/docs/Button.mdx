This is a Button

## Props

| Name         | Type          | Default value  | Description      |
|--------------|---------------|----------------|------------------|
| theme        | string        | primary        | button type      |
| size         | string        | md             | button size      |
| disabled     | bool          | false          | disable button   |
| loading      | bool          | false          | loading button   |
| icon         | string        | -              | button icon      |

### Code
```jsx
// How to import and use a Button
import Button from '@occmundial/atomic/dist/Button'

function Component() {
  return (
    <Button>Hey, click me!</Button>
  )
}
```
