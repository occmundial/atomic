# NavButton

---

The NavButton component is icon component adapted to work with with Nav components

## Props

| Name           | Type                           | Default value    | Description                                        |
| -------------- | ------------------------------ | ---------------- | ---------------------------------------            |
| `selected`     | **boolean**                    | -                | Highlights the icon and label                      |
| `iconName`     | **string**                     | -                | Icon to diplay                                     |
| `label`        | **string**                     | -                | Icon label                                         |
| `onClick`      | **EventHandler<MouseEvent\>**  | -                | Component onCLick event                            |
| `showBar`      | **boolean**                    | -                | Shows an indicator at the bottom of the component  |
| `direction`    | **'col' \| 'row'**             | 'row'            | Defines icon and label stacking direction          |
| `className`    | **string**                     | -                | Anchor className attribute                         |
| `width`        | **number**                     | -                | Defines component width                            |
| `white`        | **boolean**                    | -                | Applys white theme                                 |
| `testId`       | **string**                     | -                | Anchor test id                                     |


## Examples

You can choose between two themes: Default and white.
