# TextField

---

- [Props](#Props)
- [Examples](#Examples)

The TextField component is used to get user input. It can be used to get text, numbers, dates, etc. It can also be used to get user input from a list of options.
It supports the following types:
- any text or number input type (text, number, tel, url, etc.)
- select
- textarea
- password

You can also use custom patterns for validation.
The TextField can handle default, focus, error and disabled states.

## Props

| Name                | Type                      | Default | Description                                                                 |
| ------------------- | ------------------------- | ------- | --------------------------------------------------------------------------- |
| `type`              | **string**                | `text`  | The type of input field.                                                    |
| `maxLength`         | **number**                |         | The maximum length of the input value.                                      |
| `label`             | **string**                |         | The label for the input field.                                              |
| `placeholder`       | **string**                |         | The placeholder text for the input field.                                   |
| `assistiveText`     | **ReactNode**             |         | An assistive text shown below the input field.                              |
| `name`              | **string**                | `false` | The name attribute of the input field.                                      |
| `disabled`          | **boolean**               | `false` | Determines if the input field is disabled.                                  |
| `counter`           | **boolean**               | `false` | Determines if a character counter is displayed.                             |
| `autoFocus`         | **boolean**               | `false` | Determines if the input field should be focused automatically.              |
| `clear`             | **boolean**               | `false` | Determines if a clear button is displayed.                                  |
| `error`             | **boolean**               | `false` | Determines if the input field has an error state.                           |
| `allowError`        | **boolean**               | `false` | Determines if the error state is allowed without touching the button.       |
| `value`             | **string**                | ``      | The value of the input field.                                               |
| `selectOnFocus`     | **boolean**               | `false` | Determines if the input value should be selected by the cursor on focus.    |
| `mask`              | **any**                   |         | The mask pattern for the input field.                                       |
| `guide`             | **boolean**               | `false` | Determines if the mask guide should be displayed.                           |
| `pattern`           | **string**                |         | The custom pattern for validation.                                          |
| `inputMode`         | **'text'** &#124; **'none'** &#124; **'search'** &#124; **'tel'** &#124; **'url'** &#124; **'email'** &#124; **'numeric'** &#124; **'decimal'** |         | The input mode for the input field. It affects the keyboard layout.         |
| `disableAutoComplete` | **boolean**             | `false` | Determines if autocomplete is disabled for the input field.                  |
| `onFocus`           | **function**             |         | Event handler for when the input field is focused.                          |
| `onBlur`            | **function**             |         | Event handler for when the input field loses focus.                         |
| `onChange`          | **function**             |         | Event handler for when the input value changes.                             |
| `onKeyUp`           | **function**             |         | Event handler for key up events in the input field.                         |
| `onClear`           | **function**             |         | Event handler for when the clear button is clicked.                         |
| `options`           | **OptionProps[]**        | []      | The list of options for a select input field.                               |
| `iconName`          | **string**                |         | The name of the icon to display in the input field.                         |
| `inputClassName`    | **string**                |         | The class name for the input field.                                         |
| `regex`             | **string** &#124; **RegExp** |         | The regular expression pattern for validation.                              |
| `required`          | **boolean**               | `false` | Determines if the input field is required.                                  |
| `alignRight`        | **boolean**               | `false` | Determines if the input field's value should be aligned to the right.       |
| `id`                | **string**                |         | The id attribute of the input field.                                        |
| `className`         | **string**                |         | The class name for the input field container.                               |
| `style`             | **CSSProperties**         |         | The inline style for the input field container.                             |
| `testId`            | **string**                |         | The test id attribute for testing purposes.                                 |
