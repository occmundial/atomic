# MenuItem

---

MenuItem component with left and right icons.

## Props

Props of the native component are also available.

| Name             | Type                                                                                                                           | Default value  | Description                                                                                                                                               |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------ | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `alignItems`     | **center \| start**                                                                                                            | `center`       | Defines the align-items style property.                                                                                                                   |
| `className`      | **{ root: string; textContainer: string; iconRightContainer: string; iconLeftContainer: string }**                             | -              | Adds classes to the root, text, iconRight, iconLeft containers.                                                                                           |
| `style`          | **{ root: CSSProperties; textContainer: CSSProperties; iconRightContainer: CSSProperties; iconLeftContainer: CSSProperties }** | -              | Adds styles to the root, text, iconRight, iconLeft containers.                                                                                            |
| `nonActivatable` | **boolean**                                                                                                                    | `false`        | Use to remove hover, focus and active stylings.                                                                                                           |
| `dense`          | **boolean**                                                                                                                    | -              | If true, compact vertical padding designed for keyboard and mouse input is used. The prop defaults to the value inherited from the parent List component. |
| `disablePadding` | **boolean**                                                                                                                    | `false`        | If true, all padding is removed.                                                                                                                          |
| `disableGutters` | **boolean**                                                                                                                    | `false`        | If true, the left and right padding is removed.                                                                                                           |
| `children`       | **ReactNode**                                                                                                                  | -              | The content of the component.                                                                                                                             |
| `selected`       | **boolean**                                                                                                                    | `false`        | Use to apply selected styling.                                                                                                                            |
| `component`      | **ElementType**                                                                                                                | `div`          | The component used for the root node. Either a string to use a HTML element or a component.                                                               |
| `disableText`    | **boolean**                                                                                                                    | `false`        | If true, the children won't be wrapped by a Text component. This can be useful to render an alternative Text variant by wrapping the children text.       |
| `textProps`      | **[TextProps](/Text#Props)**                                                                                                   | -              | These props will be forwarded to the primary typography component (as long as disableText is not true).                                                   |
| `iconRightProps` | **[IconProps](/Icon#Props)**                                                                                                   | -              | These props will be forwarded to the Icon component.                                                                                                      |
| `iconLeftProps`  | **[IconProps](/Icon#Props)**                                                                                                   | -              | These props will be forwarded to the Icon component.                                                                                                      |
